# gustavo_fazolo
🇧🇷 Culturas do Brasil

Um projeto web educativo desenvolvido com HTML, CSS e JavaScript para apresentar informações sobre as diferentes culturas das regiões brasileiras.

📖 Sobre o Projeto

O site Culturas do Brasil tem como objetivo divulgar aspectos culturais das cinco regiões do país de forma simples, visual e interativa.

Além de apresentar características de cada região, o projeto inclui uma seção de curiosidades que exibe informações aleatórias sobre a cultura brasileira.

🚀 Funcionalidades
Exibição das cinco regiões do Brasil:
🌿 Norte
🎉 Nordeste
🐎 Centro-Oeste
🏙️ Sudeste
🍇 Sul
Navegação por âncoras entre as regiões.
Layout responsivo e agradável.
Botão para exibir curiosidades aleatórias sobre o Brasil.
Estilização inspirada nas cores da bandeira brasileira.

function mostrarCuriosidade() {
    const indice = Math.floor(Math.random() * curiosidades.length);

    document.getElementById("curiosidade-texto").innerText =
        curiosidades[indice];
}
